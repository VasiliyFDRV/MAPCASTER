# MAPCASTER (DnD Maps Assistant)

MAPCASTER — desktop-приложение для мастера DnD: управление сценами, отображение карты и фона, рисование и тактические пометки по ходу сессии.

## Концепция

Приложение работает с 3 окнами:

1. `Лаунчер` — библиотека приключений и сцен, создание/редактирование сцен.
2. `Окно карты` — карта сцены, гекс-сетка и инструменты взаимодействия.
3. `Окно фона` — фоновое изображение/видео для атмосферы.


## Что реализовано сейчас

### Лаунчер

- Тёмный UI (русский язык).
- Списки приключений и сцен.
- Создание/переименование/удаление приключений.
- Создание/редактирование/удаление сцен.
- Открытие сцены двойным кликом.
- Перетаскивание сцен для изменения порядка.
- Настройки открываются иконкой-шестерёнкой.

### Создание/редактирование сцены

- Компактные блоки `Карта` и `Фон`.
- Для каждого блока:
  - переключатель `On/Off`;
  - компактное превью-поле;
  - `Drag & Drop` файла;
  - `Ctrl+V` (вставка);
  - двойной клик для выбора через проводник;
  - кнопка палитры для выбора цвета.
- Настройки гекс-сетки:
  - размер клетки (ft),
  - толщина линий,
  - прозрачность,
  - цвет (включая кнопку палитры).

### Окно карты

#### Навигация

- Масштабирование карты колесом мыши.
- Перемещение карты (pan) инструментом перемещения.
- Полноэкранный режим кнопкой в панели.

#### Сетка

- Отрисовка гекс-сетки поверх карты.
- Сетка масштабируется и двигается вместе с картой.

#### Инструменты

- `Курсор`.
- `Перо`:
  - рисование свободной линией,
  - настраиваемые цвет / размер (в футах) / прозрачность.
- `Заливка`:
  - заливка областей,
  - настраиваемые цвет / прозрачность.
- `Ластик`:
  - мягкий ластик с размером (в футах) и мягкостью,
  - стирание в реальном времени.
- `Выделение гексов`:
  - выделение гекс-клеток цветом,
  - управление прозрачностью заливки и контура.
- `Измерение`:
  - измерение от центра гекса до центра гекса,
  - отображение расстояния в футах,
  - текущий масштаб: `1 гекс = 5 ft`.

#### Служебные действия

- `Undo` (отмена последнего действия).
- `Clear all` (очистка визуальных слоёв сцены).
- `Save` (ручное сохранение сцены).

### Окно фона

- Отдельный вывод фонового изображения/видео сцены.

## Хранение данных

- Приключения и сцены хранятся в файловой структуре `adventures/...`.
- Настройки приложения: `app_data/settings.json`.

## Технологии

- Python
- PySide6 / QML
- Локальная файловая модель данных (без сервера)

## Требования

- Windows
- Python `>= 3.12`

## Запуск в разработке

```powershell
python -m pip install -e .
python -m app.main
```

Альтернатива:

```powershell
run_dnd_maps.bat
```

## Структура проекта

- `app/` — логика приложения
- `app/ui/qml/` — QML-интерфейсы окон
- `app/ui/qml/components/` — переиспользуемые QML-компоненты
- `adventures/` — данные приключений/сцен
- `app_data/` — локальные настройки

## Текущее направление доработки

2. Довести поведение показа/скрытия окон карты и фона по параметрам сцены.
3. Финальная полировка UI/UX лаунчера и диалогов.

## Важно про git

В `adventures/` лежат ваши рабочие данные сцен. При коммитах лучше добавлять только кодовые изменения (`app/...`), чтобы не потерять и не перезаписать игровые данные.
