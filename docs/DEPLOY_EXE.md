# Сборка и запуск EXE на другом компьютере (Windows)

## 1) Сборка EXE

На компьютере разработчика:

1. Откройте папку проекта.
2. Запустите `build_exe.bat`.
3. Дождитесь окончания сборки.

Результат:
- `dist\DnD_MAPS\DnD_MAPS.exe`

## Быстрый portable ZIP

Если нужен сразу архив для передачи:

1. Запустите `build_portable_zip.bat`
2. Готовый файл: `dist\DnD_MAPS_portable.zip`

## 2) Что переносить на другой ПК

Переносить нужно **всю** папку:

- `dist\DnD_MAPS\`

Не только `DnD_MAPS.exe`, а весь каталог рядом с ним.

## 3) Запуск на другом ПК

1. Скопируйте `dist\DnD_MAPS\` в любое место (например, `D:\DnD_MAPS`).
2. Запустите `DnD_MAPS.exe`.

Приложение хранит данные рядом с exe:
- `app_data\settings.json`
- `adventures\...`

## 4) Быстрое обновление версии

При новой сборке замените на другом ПК старую папку `DnD_MAPS` на новую.

## 5) Типовые проблемы

- Ошибка запуска из-за отсутствия Visual C++ Runtime:
  - установите Microsoft Visual C++ Redistributable (x64).
- Антивирус блокирует запуск:
  - добавьте папку `DnD_MAPS` в исключения.
